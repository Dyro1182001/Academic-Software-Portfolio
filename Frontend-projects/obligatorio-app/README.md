# Apkionic

A web application packaged as an Android mobile application using Capacitor.

## Technologies & Tools

- HTML
- CSS
- JavaScript
- Capacitor
- Android
- Android Studio

## Project Structure

- `www/` - Contains the web application's HTML, CSS, and JavaScript files.
- `android/` - Contains the Android project generated with Capacitor.

The project's dependencies are listed in `package.json`.
