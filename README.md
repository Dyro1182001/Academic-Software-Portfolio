# Academic & Personal Software Portfolio

Welcome to my software development portfolio. This repository brings together a selection of academic and personal projects created throughout my studies and continued learning in Information Technology.

The projects included here reflect my progression as a developer, with a particular focus on backend development, databases, and full-stack applications. Rather than serving as a collection of individual coursework assignments, this repository is intended to showcase the practical skills and technical knowledge I have developed through different projects and learning experiences.

Cloud deployment and DevOps practices are not the primary focus of this portfolio. With the exception of a document from an early Hands-on Deployment Workshop completed at ORT, these areas have been intentionally left out of this repository, as I already have a dedicated repository pinned to my GitHub profile that showcases my experience and knowledge in DevOps, deployment, and related practices.

## What You'll Find Here

* **Backend Development** — Applications and APIs built with technologies such as C#/.NET, Java, Python, and PHP.
* **Frontend Development** — Web interfaces and applications using JavaScript, React, HTML, and CSS.
* **Databases** — SQL, Oracle, PostgreSQL, database design, queries, and data persistence.
* **Software Development** — Projects involving application architecture, system design, and software engineering principles.

## Repository Organization

The projects in this portfolio are organized by their primary focus:

* `Backend-projects` — Backend applications, APIs, and server-side development.
* `Fullstack-projects` — Projects combining frontend and backend components into complete applications.
* `Frontend-projects` — Web applications and projects focused primarily on user interfaces.
* `Databases` — Database design, SQL queries, data modeling, and persistence-related work.
* `Deployment` — Selected documentation and materials related to cloud deployment.
